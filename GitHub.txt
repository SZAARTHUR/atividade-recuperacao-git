O que é o GitHub ?
GitHub é um site que funciona como uma “nuvem” para guardar projetos feitos com Git.
Ele não substitui o Git — na verdade, ele depende do Git. Enquanto o Git cuida das versões localmente, o GitHub é o lugar onde você envia o seu projeto para ficar online, compartilhar com outras pessoas, fazer backup e trabalhar em equipe. 
Em resumo: Git é a ferramenta de controle de versão no seu computador, e GitHub é o lugar na internet onde você armazena e organiza esses projetos.
